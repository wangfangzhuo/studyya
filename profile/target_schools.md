# 目标院校研究

## ⚡ 核心优势：管理类联考只考2门

无论报 MBA / MPA / MPAcc，考试科目完全相同：
- **英语二**（100分）
- **管理类综合199**（200分）：数学基础 + 逻辑 + 写作

**没有政治！** 普通考研要考政治(100分)，管理类联考不需要。
→ 这意味着：你只需备考2门，精力高度集中，对在职人员极为友好。

备考策略与报考院校/专业无关，**先全力备考，考完再报志愿。**

## MBA vs MPA：你最需要了解的区别

| 对比项 | MBA | MPA |
|--------|-----|-----|
| 全称 | 工商管理硕士 | 公共管理硕士 |
| 考试 | 完全相同 | 完全相同 |
| 学历证书 | 相同（硕士） | 相同（硕士） |
| 学位证书 | MBA学位 | MPA学位 |
| 学费 | 贵（10-28万） | 便宜（5-12万） |
| 适合人群 | 企业管理方向 | 政府/事业单位/非营利 |
| 含金量 | 商界认可度高 | 公共部门认可度高 |

**结论：** 如果你在企业工作且目标是升职/跳槽，MBA更对口；
如果费用是核心考量，同济/华东师大的MPA是极高性价比选择。
