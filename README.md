## 每日流程

1. 打开当天的 `plans/daily/YYYY-MM-DD.md`
2. 完成学习内容（趣味化输出）
3. 做当日测验题并填写答案
4. 填写今日总结
5. 更新 `progress/knowledge_map.md`（如有新进展）
