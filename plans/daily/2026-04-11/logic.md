## 一、全称命题与演绎推理

### 什么是全称命题？

"**所有**A都是B"、"**凡是**A都是B"、"**任何**A都是B"——这类句子叫全称命题。

用符号表示：对所有x，若x是A，则x是B。

### 演绎推理怎么用？

```
前提1：所有 A 都是 B
前提2：C 是 A
结论 ：C 是 B  ← 一定成立
```

**你在摸底里做对的那道题就是这个：**

> 所有参加会议的人都是经理（所有A是B）
> 张三参加了会议（C是A）
> → 张三是经理（C是B）✅

### 常见陷阱：方向反了

```
前提1：所有 A 都是 B
前提2：C 是 B
结论 ：C 是 A ← ❌ 不一定成立！
```

举例：
> 所有猫都是动物（所有A是B）
> 狗是动物（C是B）
> → 狗是猫？❌ 明显错误

**方向错了，结论就错了。** 这是形式逻辑最基础的陷阱。

---

## 二、充分条件初识（下周要深挖的核心）

全称命题"所有A是B"，其实可以改写成：

> **"如果是A，那么是B"** → A → B

这里，A 是 B 的**充分条件**——有A就够了，B一定出现。

**生活例子：**

- "如果下雨，地面就会湿" → 下雨(A) → 地面湿(B)
  - 下雨了，地面一定湿 ✅
  - 地面湿了，不一定是因为下雨（可能有人浇了水）❌

"下雨"对"地面湿"来说是**充分条件**：有它足够，但它不是唯一原因。

---

### 今天先记住这一条规则

> **肯定前件必肯后件：已知 A → B，已知 A 发生，则 B 一定发生。**

至于"已知B发生，能推A吗？"、"已知A没发生，能推B没发生吗？"——这些情况明天专门讲透。

## 解析

**题1答案：B（不能确定）**
"所有注册会员都能享受折扣" = 注册会员 → 享受折扣
已知"享受了折扣"（B成立），推"是注册会员"（A成立）→ 逆肯式，无效。
也许商家给了特殊优惠，小陈不是会员也享受到了。

**题2答案：A（能）**
"只要提交申请，就收到回执" = 提交(A) → 收到回执(B)
已知"提交了申请"（A成立）→ 顺肯式，B一定成立。

**题3答案：A（能）**
努力(A) → 考上(B)，已知¬B（没考上）→ 逆否推出¬A（没努力）
→ 这是逆否推理，Day 2 会彻底讲透。
