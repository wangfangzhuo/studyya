## 核心公式

```
涨价 x%  →  新值 = 原值 × (1 + x%)
降价 x%  →  新值 = 原值 × (1 - x%)
打 n 折  →  新值 = 原值 × (n ÷ 10)
```

**黄金技巧：遇到连续变化，直接连乘，不要逐步加减。**

---

## 概念辨析：两个容易混的说法

| 说法 | 意思 | 例子 |
|------|------|------|
| 增长了 x% | 在原来基础上增加了 x% | 100增长了20% → 120 |
| 增长到 x% | 直接变成原来的 x% | 100增长到120% → 120 |
| 是原来的 x 倍 | 变成原来的 x 倍 | 100是原来的1.5倍 → 150 |
| 比原来多 x 倍 | 变成原来的 (1+x) 倍 | 100比原来多0.5倍 → 150 |

注意最后两行——"是原来的1.5倍"和"比原来多1.5倍"结果完全不同！

## 例题精讲

**例1（原题）：** 某商品原价200元，先涨价20%，再打八折，最终售价是多少？

```
200 × 1.2 × 0.8
= 200 × 0.96
= 192元
```

注意：1.2 × 0.8 = 0.96，不是1.0。涨20%再打八折，最终比原价**便宜了4%**。

**例2：** 某商品原价500元，先降价10%，再涨价10%，最终售价是多少？

```
500 × 0.9 × 1.1
= 500 × 0.99
= 495元
```

降10%再涨10%，最终比原价**便宜了1%**，不是回到原点。

---

## 规律总结

**先涨 a%，再降 a%（相同幅度）：**
```
最终 = 原值 × (1+a%) × (1-a%) = 原值 × (1 - a²%)
```

例：±20% → 1.2 × 0.8 = 0.96，比原价少4%（= 20² ÷ 100 = 4%）
例：±10% → 1.1 × 0.9 = 0.99，比原价少1%（= 10² ÷ 100 = 1%）

**结论：相同幅度先涨后降（或先降后涨），结果永远比原价低。**
