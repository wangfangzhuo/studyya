## 核心图解：P→Q 到底是什么形状？

"只要P，就Q" = P → Q

画成圈圈图是这样的：

```
  ┌─────────────── Q 的范围 ───────────────┐
  │                                        │
  │      ┌──── P 的范围 ────┐              │
  │      │                  │   其他原因   │
  │      │   P 在 Q 里面    │   也能导致Q  │
  │      └──────────────────┘              │
  │                                        │
  └────────────────────────────────────────┘
             Q 的外面 = ¬Q
```

**读懂这张图，四种推理全部解决：**

| 已知条件 | 推理 | 结论 | 有效？ | 原因 |
|----------|------|------|--------|------|
| 某人在 P 圈里 | → | 一定在 Q 圈里 | ✅ | P 整个在 Q 里 |
| 某人**不在** Q 圈里 | → | 一定**不在** P 圈里 | ✅ | P 整个在 Q 里，Q外面不可能有P |
| 某人在 Q 圈里 | → | 不知道是不是在 P 里 | ❌ | Q 里有P，也有别的 |
| 某人**不在** P 圈里 | → | 不知道是不是在 Q 里 | ❌ | 不是P，可能还有别的原因进入Q |

## 用你答错的两道题验证

### 题1（你答了A，正确是B）

> "所有注册会员都能享受折扣。小陈享受了折扣。"

画图：
```
  ┌──── 享受折扣(Q) ────┐
  │  ┌──会员(P)──┐      │
  │  │           │ ←小陈在这里？  │
  │  └───────────┘ 还是这里？│
  └────────────────────┘
```

小陈享受了折扣，他在 Q 圈里。  
但 Q 圈里有 P（会员），也有别的（特别促销、员工价…）。  
**不知道小陈在哪一块，答案 B（不能确定）。**

---

### 题3（你答了C，正确是A）

> "只要努力学习，就能考上研究生。小李没有考上研究生。"

画图：
```
  ┌──── 考上研究生(Q) ────┐
  │  ┌── 努力学习(P) ──┐  │
  │  │                  │  │
  │  └──────────────────┘  │
  └──────────────────────── ┘
         Q外面 = 没考上 = ¬Q
```

小李没考上，他在整个大圈的**外面**（¬Q 区域）。  
P 圈整个在 Q 圈里面，所以 ¬Q 的区域里**绝对没有 P**。  
→ 小李不可能在 P 圈里 → 小李没有努力学习。  
**答案 A（能推出）。**

## 你的直觉为什么会错？

你选 C 的原因很可能是：**"现实中努力了不一定考上啊，这个前提本身就不成立。"**

这是形式逻辑最关键的规则：

> **逻辑题里，前提永远被当作真的。**
> 我们只判断"推理过程是否有效"，不评判"前提本身对不对"。

就像玩象棋，车只能走直线是规则，不能因为觉得不合理就不遵守。  
逻辑推理也是：题目给了前提，就当它是真的，看推理步骤对不对。

## 记忆总结：两句话

```
✅ 肯前推后：已知P，推Q（顺着箭头走）
✅ 否后推前：已知¬Q，推¬P（箭头反向，符号也反）

❌ 肯后推前：已知Q，推P（不能反向走）
❌ 否前推后：已知¬P，推¬Q（不能反向走）
```

**最简口诀：顺着走、反着走都行；只有顺着走不行。**  
（顺肯✅，逆否✅；逆肯❌，否前❌）
