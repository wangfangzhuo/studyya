> 今日专题：**比与比例——从切蛋糕到按比分配**

---

## 先聊聊为什么要学这个

考研数学里有一类题叫"按比例分配"，非常高频：

> "甲乙丙三人按 2:3:5 的比例分配奖金10万元，各得多少？"

看起来简单，但考试里会加很多变体（比例变化、总量未知、部分已知求其他），搞清楚原理才能应对所有变体。

---

## 一、比的基本概念

**比 a:b** 表示 a 和 b 的倍数关系。

```
a:b = a/b（本质就是分数）
```

**化简比：** 找最大公因数，约分。

- 12:18 = 2:3（除以6）  
- 0.4:0.6 = 4:6 = 2:3（先去小数点）  
- 1/3 : 1/4 = 4:3（通分后比较分子）

---

## 二、按比例分配（核心题型）

**方法：先求"1份是多少"，再乘以各自的份数。**

**例1：** 甲乙丙按 2:3:5 分 10000 元，各得多少？

```
总份数 = 2 + 3 + 5 = 10 份
1份 = 10000 ÷ 10 = 1000 元
甲 = 2 × 1000 = 2000 元
乙 = 3 × 1000 = 3000 元
丙 = 5 × 1000 = 5000 元
```

验证：2000 + 3000 + 5000 = 10000 ✅

---

## 三、常见变体

### 变体1：已知一份，求总量或其他份

**例2：** 甲乙按 3:4 分钱，甲得了 600 元，乙得了多少，总共多少？

```
甲是3份 = 600 → 1份 = 200 元
乙是4份 = 200 × 4 = 800 元
总计 = 600 + 800 = 1400 元
```

### 变体2：比例改变

**例3：** 甲乙原来按 2:3 分，后来甲多了200元，比例变成 3:4，总金额不变，原来总金额是多少？

设原总金额为 x：
```
原来甲得：x × 2/5
后来甲得：x × 3/7
差值：x × 3/7 - x × 2/5 = 200
     x × (15/35 - 14/35) = 200
     x × 1/35 = 200
     x = 7000 元
```

---

## 四、比例的基本性质

**比例** a:b = c:d，等价于 **a × d = b × c**（交叉相乘相等）

用途：已知三个量，求第四个。

**例4：** 5:8 = x:24，求 x

```
5 × 24 = 8 × x
120 = 8x
x = 15
```
